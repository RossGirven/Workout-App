Cloud setup

  1. Create a Supabase project.
  2. Run the SQL in supabase-setup.sql.
  3. Copy config.example.js to config.js and paste your project URL and anon key.
  4. In Supabase Auth, enable email sign-in with password.
  5. Upload the whole folder to GitHub Pages.
  6. Use the same email on each device so the cloud copy can be loaded anywhere.

Use the anon key in the browser, never the service role key. Exposing that one would be a truly inspired disaster.
